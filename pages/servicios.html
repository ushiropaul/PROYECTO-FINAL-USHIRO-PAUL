Servicios de Ingeniería Social:

Estas habilidades se centran en manipular psicológicamente a las personas para obtener información confidencial o acceso a sistemas.
Esto puede implicar el engaño a través de correos electrónicos, llamadas telefónicas u otros medios de comunicación.

Ingeniería social

Implica manipular psicológicamente a las personas para obtener información confidencial o acceso a sistemas, a menudo a través de engaños en correos electrónicos, llamadas telefónicas u otros medios de comunicación.

Phishing

Es una técnica de ingeniería social que utiliza correos electrónicos, mensajes de texto o sitios web fraudulentos para engañar a las personas y obtener información confidencial, como contraseñas o detalles de tarjetas de crédito.

Servicios de Explotación de Software y Sistemas:

Este grupo se refiere a habilidades para identificar y aprovechar vulnerabilidades en software y sistemas informáticos con el fin de obtener acceso no autorizado o causar daño.

Explotación de vulnerabilidades

Implica identificar y aprovechar debilidades en software y sistemas informáticos para obtener acceso no autorizado o causar daño.

Inyección de Código.

Consiste en insertar código malicioso en una aplicación web o sistema para ejecutar comandos no autorizados o robar información confidencial.

Habilidades de Ataques de Red:

Estas habilidades se utilizan para interceptar y manipular el tráfico de red, comprometiendo la seguridad de los datos transmitidos.

DoS/DDoS

Son ataques que intentan saturar un servidor o red con una cantidad abrumadora de solicitudes, lo que provoca que el sistema sea inaccesible para usuarios legítimos.

Sniffing de Red

Es la práctica de interceptar y monitorear el tráfico de red para capturar información sensible, como contraseñas o datos financieros, transmitidos sin cifrar.

Spoofing

Implica falsificar la identidad de una entidad, como una dirección IP o un sitio web, para engañar a los usuarios y obtener acceso no autorizado o realizar ataques de phishing.

Servicios de Acceso y Recolección de Datos:

Este grupo involucra habilidades para acceder a información confidencial, como contraseñas, y para recopilar datos sin autorización.

Keylogging

Es la acción de registrar las pulsaciones de teclas realizadas por un usuario en un dispositivo, lo que permite al atacante obtener contraseñas u otra información confidencial.

Fuerza Bruta

Es un método de prueba que consiste en probar sistemáticamente todas las combinaciones posibles de contraseñas hasta encontrar la correcta para obtener acceso no autorizado a un sistema o cuenta.

Man-in-the-Middle (MitM)

Se refiere a un ataque en el que un atacante intercepta y manipula la comunicación entre dos partes, lo que le permite robar información confidencial o modificar datos.

Servicios de Ataques a la Seguridad de Aplicaciones:

Estas habilidades se enfocan en la explotación de vulnerabilidades en aplicaciones web, móviles u otros sistemas para obtener acceso no autorizado o causar daño.

Ataques de Intermediario Web (Web Proxy)

Consiste en manipular el tráfico entre un usuario y un servidor web, a menudo para robar información confidencial o realizar ataques de tipo Man-in-the-Middle.

Ataques de Inyección de SQL

Consiste en manipular el tráfico entre un usuario y un servidor web, a menudo para robar información confidencial o realizar ataques de tipo Man-in-the-Middle.

Ataques de Cross-Site Scripting (XSS)

Se refiere a la inserción de scripts maliciosos en páginas web visitadas por otros usuarios, lo que puede conducir al robo de cookies de sesión o la ejecución de acciones no autorizadas en el navegador del usuario.

Servicios de Ataques a la Infraestructura de TI:

Este grupo incluye habilidades para comprometer y manipular la infraestructura de TI, como servidores y sistemas embebidos.

Ataques a Sistemas Embebidos

Involucra la explotación de vulnerabilidades en dispositivos embebidos, como cámaras de seguridad o dispositivos IoT, para obtener acceso no autorizado o realizar ataques.

Ataques de Falsificación de Certificados SSL/TLS

Consiste en falsificar un certificado SSL/TLS para interceptar y manipular la comunicación cifrada entre un usuario y un servidor, permitiendo al atacante leer o modificar datos sensibles.
